# Food-Web-App

Built a responsive food ordering web application using React.js and Swiggy API integration. The application dynamically fetches and displays restaurant data, including restaurant names, cuisines, ratings, and delivery details. Implemented features such as restaurant search, dynamic routing, reusable components, shimmer UI for loading states, and real-time data rendering using React hooks like useState and useEffect. The project focuses on component-based architecture, API handling, and improving frontend performance and user experience.
